# 4D WeatherAPI

## Overview

This 4D component provides:

A WeatherAPI wrapper class to access real-time weather and astronomy data from WeatherAPI.com

An AIKit tools definition class to expose weather functions as tools usable by 4D AIKit

It is designed to make weather data easily accessible from both standard 4D code and AI-driven workflows.

## Requirements

4D v20 or higher (with support for classes and 4D.HTTPRequest)

A valid API key from https://www.weatherapi.com/

4D AIKit (for AI tool integration)
